HEIC — High Efficiency Image Container

HEIC is a file extension used by Apple for images in the HEIF (High Efficiency Image Format), usually compressed with the HEVC (High Efficiency Video Compression) codec.

This format works perfectly for Apple's live photos, as it can contain sequences of images.

The biggest advantage is storage efficiency: you can keep twice as many photos in HEIC compared to JPEG at the same size and quality, thanks to next-generation compression.

JPG — Joint Photographic Experts Group

The JPG extension is assigned to image files. It's one of the most common formats used for photographs and web graphics.

To reduce file size, raster images are stored in .jpg, which makes them easier to share and upload online.

JPG is based on a 24-bit color palette. The stronger the compression applied during file creation, the more visible the loss of quality when the image is uncompressed.
